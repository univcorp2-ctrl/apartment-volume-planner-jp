# Product roadmap

## Phase 1 — shipped in this repository

- Rectangular or metric-GeoJSON site input
- BCR/FAR/road-width/height/setback calculations
- Approximate floor and unit count
- Stair/corridor/frontage review flags
- Browser-generated SVG schematic plan
- Tests, CI, devcontainer and static deployment

## Phase 2 — parcel intelligence

- Address and map search
- Parcel boundary editing and road-edge attribution
- MLIT API, National Land Numerical Information and PLATEAU ingestion
- Source freshness, confidence and missing-data dashboard
- Tokyo 23 wards as the first curated jurisdiction

## Phase 3 — regulatory envelope

- Multiple zoning polygons and weighted BCR/FAR
- Road, adjacent-lot and north-side slant planes
- Height districts, absolute-height limits, fire zones and district plans
- Reverse-shadow and sky-factor analysis
- Buildable 3D “birdcage” export

## Phase 4 — apartment layout solver

- Core count and placement
- Stairs, elevators, shafts, corridors and refuge paths
- Unit mix, frontage, daylight and ventilation constraints
- Parking, bicycle and waste-storage layouts
- Multi-objective optimization: sellable/rentable area, cost, daylight and constructability

## Phase 5 — professional workflow

- Architect rule review and exception handling
- DXF/IFC/Revit handoff
- Cost and rental pro forma
- Permit pre-check evidence package
- Organization permissions, audit trail and rule-pack subscriptions
